骇客的态度

  1. 世界是有大量有趣的问题是等待解决的, # 自发性的解决问题;
  2. 没有人应该有需要解决同样问题两次, # 不要重复发明轮子;
  3. 拒绝沉闷及乏味的事情, # 将所有沉闷及乏味的事情尽量自动化;
  4. 自由,
  5. 信念不能替代能力, # 有信念也不能, 必须付出智慧,锻炼及勤奋;

基本骇客技能

1．学如何写电脑程式, # 建议学习Python哦.. 2．取得一份开放程式码的unix作业系统, 然后学习它, # 安装linux/BSD 3. 学习如何用互联网及用HTML来写网页,

骇客社群的身份

  1. 编写公开程式码的软件;
  2. 帮助他人测试及除错公开程式码的软件;
  3. 发行有用的资讯;
  4. 帮助系统运作;
  5. 为骇客社群服务;
